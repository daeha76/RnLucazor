Features

Everything you need to use beautiful icons in your .NET applications

🔍

Full IntelliSense

Every icon is a strongly-typed static property. Get autocomplete and compile-time validation with RnIcons.Activity, RnIcons.Heart, etc.

📦

Zero Dependencies

The core library targets netstandard2.0 with absolutely no external dependencies. Works everywhere .NET runs.

⚡

Blazor Component

Drop-in <RnIcon> component with DI support. Supports size, color, stroke width, transforms, and CSS classes.

🔄

Icon Transforms

Built-in rotation (90/180/270) and flip (horizontal/vertical) transforms. Custom rotation angles also supported.

🔧

Pluggable Rendering

Implement ISvgRenderer to render icons to any target: WPF, MAUI, SkiaSharp, PDF, and more.

🧪

DI & Testing

IIconProvider interface makes it easy to inject icons and mock them in unit tests. Register custom icons alongside built-in ones.

Quick Start

Get up and running in minutes

C# / .NET
Blazor
Advanced
using RnLucazor;

// Static access — compile-time safe, with full IntelliSense
var svg = RnIcons.Activity.ToSvg();
var svg2 = RnIcons.ArrowRight.ToSvg(size: 32, stroke: "red");

// Dynamic lookup by name (auto-initializes on first call)
var icon = RnIconRegistry.GetIcon("activity");
Console.WriteLine(icon?.ToSvg());

// List all available icons
foreach (var name in RnIconRegistry.GetIconNames())
{
    Console.WriteLine(name);
}
@* Register in Program.cs *@
builder.Services.AddRnLucazor();

@using RnLucazor
@using RnLucazor.Blazor

<RnIcon Icon="RnIcons.Activity" />

<RnIcon Name="arrow-right" Size="16" />
<RnIcon Icon="RnIcons.ArrowUp" Transform="IconTransform.Rotate90" />

    Size = 48,
    Stroke = "#ff0000",
    Fill = "pink",
    StrokeWidth = 1.5f,
    CssClass = "icon-heart",
    Transform = IconTransform.Rotate90,
});

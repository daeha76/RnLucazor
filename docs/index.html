Features

Everything you need to use beautiful icons in your .NET applications

🔍

Full IntelliSense

Every icon is a strongly-typed static property. Get autocomplete and compile-time validation with RnIcons.Activity, RnIcons.Heart, etc.

📦

Zero Dependencies

The core library targets netstandard2.0 with absolutely no external dependencies. Works everywhere .NET runs.

⚡

Blazor Component

Drop-in <RnIcon> component with DI support. Supports size, color, stroke width, transforms, and CSS classes.

🔄

Icon Transforms

Built-in rotation (90/180/270) and flip (horizontal/vertical) transforms. Custom rotation angles also supported.

🔧

Pluggable Rendering

Implement ISvgRenderer to render icons to any target: WPF, MAUI, SkiaSharp, PDF, and more.

🧪

DI & Testing

IIconProvider interface makes it easy to inject icons and mock them in unit tests. Register custom icons alongside built-in ones.

Usage Guide

Step-by-step guide to using RnLucazor in your project

1 Basic SVG Output

Any .NET app can use RnIcons to get SVG strings. Compile-time safe with full IntelliSense.

C#
using RnLucazor;

// Static access — compile-time safe, with full IntelliSense
var svg = RnIcons.Activity.ToSvg();

// With inline parameters: size, stroke color, stroke width
var svg2 = RnIcons.ArrowRight.ToSvg(size: 32, stroke: "red", strokeWidth: 1.5f);

// Dynamic lookup by kebab-case name (auto-initializes on first call)
var icon = RnIconRegistry.GetIcon("arrow-right");
Console.WriteLine(icon?.ToSvg());

// List all 1700+ available icon names
foreach (var name in RnIconRegistry.GetIconNames())
    Console.WriteLine(name);

2 Advanced SVG Rendering

SvgRenderOptions gives you full control over size, color, fill, stroke, CSS class, custom attributes, and transforms.

C#
var svg = RnIcons.Heart.ToSvg(new SvgRenderOptions
{
    Size          = 48,
    Stroke        = "#ff0000",
    Fill          = "pink",
    StrokeWidth   = 1.5f,
    StrokeLineCap = "round",    // default: "round"
    StrokeLineJoin= "round",    // default: "round"
    CssClass      = "icon-heart",
    Transform     = IconTransform.Rotate90,
    Attributes    = new Dictionary<string, string>
    {
        { "data-testid", "heart-icon" },
        { "aria-hidden", "true" },
    }
});

3 Icon Transforms

Rotate and flip icons with built-in presets or custom angles. Transforms apply via SVG transform attribute.

C#
// Built-in presets
RnIcons.ArrowRight.ToSvg(new SvgRenderOptions { Transform = IconTransform.Rotate90 });
RnIcons.ArrowRight.ToSvg(new SvgRenderOptions { Transform = IconTransform.Rotate180 });
RnIcons.ArrowRight.ToSvg(new SvgRenderOptions { Transform = IconTransform.Rotate270 });
RnIcons.ArrowRight.ToSvg(new SvgRenderOptions { Transform = IconTransform.FlipHorizontal });
RnIcons.ArrowRight.ToSvg(new SvgRenderOptions { Transform = IconTransform.FlipVertical });

// Custom angle + flip combination
var transform = new IconTransform { Rotate = 45, FlipX = true };
RnIcons.Star.ToSvg(new SvgRenderOptions { Transform = transform });

5 Dependency Injection

IIconProvider interface allows DI-based icon access and easy unit testing with mock providers.

C#
public class MyService
{
    private readonly IIconProvider _icons;

    public MyService(IIconProvider icons) => _icons = icons;

    public string GetIconSvg(string name)
    {
        var icon = _icons.GetIcon(name);
        return icon?.ToSvg() ?? "";
    }

    public int TotalIcons => _icons.Count;

    public IEnumerable<string> AllNames()
        => _icons.GetIconNames();
}

6 Custom Icons

Register your own SVG icons alongside the 1700+ built-in ones using AddRnLucazor callback or RnIconProvider.AddIcon.

C# — Blazor DI Registration
using System.Collections.ObjectModel;

builder.Services.AddRnLucazor(provider =>
{
    provider.AddIcon("my-logo", new IconData("my-logo", new IconElement[]
    {
        new IconElement("circle", new ReadOnlyDictionary<string, string>(
            new Dictionary<string, string>
            {
                { "cx", "12" }, { "cy", "12" }, { "r", "10" }
            }))
    }));
});

// Now use it like any other icon:
// <RnIcon Name="my-logo" />
C# — Programmatic (non-Blazor)
var provider = new RnIconProvider();
provider.AddIcon("custom-star", myIconData);

// Lookup includes both built-in and custom icons
var icon = provider.GetIcon("custom-star");
var builtIn = provider.GetIcon("heart");

// Remove a custom icon
provider.RemoveIcon("custom-star");

7 Custom ISvgRenderer

Implement ISvgRenderer to render icons to any target platform: WPF, MAUI, SkiaSharp, PDF, etc.

C#
// Implement the interface for your target platform
public class WpfSvgRenderer : ISvgRenderer
{
    public string Render(IconData icon, SvgRenderOptions? options = null)
    {
        var opts = options ?? SvgRenderOptions.Default;
        // Access icon.Elements to get SVG child elements
        // Each element has: Tag (path, circle, rect, etc.)
        //                    Attributes (d, cx, cy, r, etc.)
        foreach (var el in icon.Elements)
        {
            var tag = el.Tag;              // e.g., "path", "circle"
            var attrs = el.Attributes;     // e.g., { "d": "M12 2..." }
        }
        return "<!-- XAML output -->";
    }
}

// Use directly
ISvgRenderer renderer = new WpfSvgRenderer();
var output = renderer.Render(RnIcons.Star, new SvgRenderOptions { Size = 16 });

// Or register via DI
builder.Services.AddSingleton<ISvgRenderer, WpfSvgRenderer>();
